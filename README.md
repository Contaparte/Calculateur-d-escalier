# Calculateur d’escalier

## Description
Ce projet est un calculateur d’escalier permettant de déterminer les dimensions principales d’un escalier résidentiel à partir de paramètres standards (hauteur à franchir, nombre de marches, giron, contremarche, etc.).

L’objectif est de fournir des calculs cohérents, reproductibles et conformes aux règles usuelles de conception d’escaliers.

---

## Fonctionnalités principales
- Calcul du nombre de marches
- Calcul de la hauteur de contremarche
- Calcul du giron
- Vérification de la pente de l’escalier
- Validation des dimensions selon des critères usuels

---

## Structure du projet
Le projet est organisé de manière modulaire :

- Les fichiers de calcul contiennent la logique mathématique et les règles métier
- Les fichiers de configuration définissent les paramètres d’entrée
- Les fichiers de présentation (si présents) gèrent l’affichage ou l’interface utilisateur

---

## Logique de calcul (vue d’ensemble)
Les calculs reposent principalement sur :
- la hauteur totale à franchir
- la relation entre giron et contremarche
- les proportions ergonomiques usuelles

Les valeurs sont ajustées afin d’obtenir un escalier fonctionnel et sécuritaire.

---

## Cas d’utilisation
Ce projet peut être utilisé pour :
- vérifier la faisabilité d’un escalier
- comparer différentes configurations
- servir de base à un outil d’aide à la conception
- alimenter un assistant intelligent de calcul et de validation

---

## Limitations
- Le projet ne remplace pas une validation professionnelle
- Certaines normes locales peuvent ne pas être intégrées
- Les résultats doivent être interprétés dans un contexte réel

---

## État du projet
Projet en cours de développement.  
Les fonctionnalités et la structure peuvent évoluer.
